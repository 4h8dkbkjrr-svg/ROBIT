# Claude 사용에 관한 안내

## 1. 배경

C언어 진도가 포인터 단원을 넘어서면서 과제의 코딩 난이도가 크게 상승

Claude를 전혀 사용하지 않고 인터넷 조사와 수업 자료 확인만으로 과제 한 개를 수행할 경우 약 3시간 정도가 소요되며, 난이도가 높아지고 구현해야 할 함수의 개수가 늘어날수록 소요 시간은 더욱 길어짐.

따라서 불가피하게 Claude를 사용하되, 그 사용 방식과 사용 내역을 투명하게 공개하기로 결정

## 2. 기본 원칙

1. 코드 초안과 로직 설계는 스스로 작성하고 수행을 시도
    - 문제 분석, 알고리즘 설계, 함수 분리, 첫 구현은 직접 진행
2. Claude는 원칙적으로 "질문"의 용도로만 사용
    - 개념이 이해되지 않는 부분, 문법의 동작 원리, 오류 메시지의 의미 등을 묻는 데 사용
3. "코드를 대신 작성해 달라"는 식의 요청은 지양
    - 완성된 답을 받아오는 방식의 사용 자제
4. 예외: 디버깅 과정에서 스스로 오류를 찾지 못하는 경우에 한해, Claude가 코드 작성에 직접 개입하는 것을 허용
    - 이 경우에도 개입 범위를 최소화하고, 주석으로 기록을 남김

## 3. 사용 내역 공개 방식

AI의 도움을 받은 부분은 소스 코드 내 주석을 통해 해당 위치에 직접 표기

1. Claude가 함수에 광범위하게 개입한 경우, `[1]` 함수 원형 선언에서 Claude가 개입했음을 표기
2. Claude가 함수에 제한적으로 개입한 경우, 각 함수에서 Claude가 개입한 줄 옆에 주석처리로 Claude가 개입했음을 표기
3. Claude가 미세하게 개입한 경우, 개입 내역이 생략될 수 있음
4. 이전 과제에서 Claude가 개입한 부분을 재사용하거나, 부분 수정해서 사용하는 경우, 개입 내역이 생략될 수 있음
5. 일부 개입 내역의 경우, 누락될 수 있음
